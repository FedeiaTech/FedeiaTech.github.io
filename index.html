Funcionalidades Básicas
Las siguientes son funcionalidades incluídas en todos los planes: galerías de imágenes, mapas de google, formulario de contacto, integración con redes sociales.


Diseños Responsivos
Diseños adaptativos que se pueden visualizar y navegar de manera amigable desde cualquier dispositivo móvil: notebooks, tablets, celulares, etc.


Páginas Web optimizadas
Trabajamos con plantillas editadas para tener la mejor performance en nuestros sitios. Conseguimos mediciones muy buenas en Google Page Speed que se tienen en cuenta para SEO.


Sitios Autoadministrables
Se pueden modificar textos e imágenes y cargar nuevos contenidos de manera sencilla sin necesidad de pagar mantenimiento. Entregamos un instructivo que explica como administrar el sitio.


Mantenimiento Web
Si bien los sitios son autoadministrables, también ofrecemos el servicio de mantenimiento web en caso de solicitarlo. Ya sea para actualizar el sistema, hacer alguna modificación o simplemente cargar contenido. No tenemos un costo fijo de mantenimiento mensual si no que trabajamos por hora.


Gestión de hosting y dominio
Podemos registrar dominios .com.ar o .com y también un servicio de hosting. Este servicio lo tercerizamos con empresas recomendadas con las que venimos trabajando muy bien en los últimos años.


Estadísticas de visitas
Instalamos los scripts y pixeles para medir accesos y visitas al sitio a través de Google Analytics. Tendrás acceso a estadísticas para conocer desde dónde acceden los usuarios que visitan el sitio, lugar de procedencia, momento del día con más tráfico, etc.


Posicionamiento SEO
Este es un servicio adicional que consiste en asociar cada página o contenido del sitio a potenciales frases de búsqueda. Es un trabajo a mediano / largo plazo que permite recibir visitas de manera orgánica sin tener que invertir en publicidad paga.
