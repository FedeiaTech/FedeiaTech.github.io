Glosario de etiquetas generales

Como insertar Tabulador en HTML
No existe una etiqueta ni un código HTML para insertar un Tab, pero puedes utilizar el código nbsp. Solo introduce 4 o 5 códigos nbsp, uno a continuación del otro.
No olvides colocar & delante de nbsp y ; al final.

Cómo agregar saltos de línea en HTML
Solo introduce tantas etiquetas < br > (sin espacios) como saltos de línea quieras insertar. No es necesario utilizar la etiqueta de cierre.

Mostrar etiquetas HTML < > sin que sean interpretadas por el navegador
Cambiamos <
por &lt;
Y >
por &gt;

¿Cómo lograr mostrar &lt?
Sencillo, escribiendo &amp;lt

Para mostrar otros caracteres especiales en HTML:
& (ampersand) por &amp;
" (comillas) por &quot;
' (apóstrofe) por &apos;
(espacio en blanco) por &nbsp;

<html> </html>   Para que el contenido sea interpretado como html
<head> </head>   Informacion sobre el documento que no se renderiza. Comunmente tags de codigos js o css
<title> </title>   Titulo de la pagina (tab del browser)
<body> </body>   Cuerpo de la pagina. sector visible
<ul> </ul>   Representa una lista desordenada
<ol> </ol>   Representa una lista ordenada
